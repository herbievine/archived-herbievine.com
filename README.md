## herbievine.com

### Contributing

Clone the repo:

```
$ git clone https://github.com/herbievine/herbievine.com.git
```

Contribute towards the website:

```
$ cd herbievine.com
$ npm i # Install dependencies
$ npm run serve # Local server
$ npm run build # Build for production
```

Contribute towards the cloud functions:

```
$ cd herbievine.com/functions
$ npm i # Install dependencies
$ npm run shell # Interactive shell to test
$ npm run build # Builds TypeScript
```
